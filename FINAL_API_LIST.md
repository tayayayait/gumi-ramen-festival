# 🚀 원평 누들-AI (K-Ramen Edition) MVP

## 최종 필수 외부 API 리스트 (정부지원사업 심사용 Web MVP 기준)

우리가 방금 합의한 **"빠른 MVP 및 스마트폰/PC 웹 링크 2개 동시 대응(React + PWA웹앱)"** 아키텍처를 100% 구현하기 위해 실제로 코딩에 들어가야 할 외부 API 최종 목록입니다.
복잡한 Unity 3D용 API들은 모두 제외하고, 순수 웹 기술로 대체 가능한 핵심 API들로만 압축했습니다.

### 1. 🤖 Google Gemini API (생성형 AI) - [연동 완료 ✅]

- **용도**: 상인이 선택한 기성 라면 + 유저가 모은 구미 특산물을 조합하여 "인스타 감성의 커스텀 레시피와 스토리" 생성
- **사용처**: `AIAdvisorPage.tsx` 및 백엔드 `Supabase Edge Function` 내부
- **상태**: 로직 작성 완료

### 2. 🗺️ Kakao Map API (2D 지도 및 위치 스폰) - [연동 완료 ✅]

- **용도**: 상인용 어드민에서 직관적인 2D 지도를 보며 AR 아이템(계란, 파 등)의 위도/경도를 클릭하여 뿌림(Spawn)
- **이유**: 상인들은 무거운 3D 맵보다 로딩이 빠르고 주소 검색이 정확한 국내 지도가 훨씬 관리하기 편합니다.
- **사용처**: `ARSpawnPage.tsx` (웹 어드민 대시보드)
- **상태**: 기본 지도 생성 및 클릭 시 마커 표시 로직 구축 완료

### 3. 🗄️ Supabase API (BaaS / 백엔드 인프라) - [연동 완료 ✅]

- **용도**: 유저 로그인(Auth), 아이템 및 레시피 데이터 저장(PostgreSQL), 반경 내 아이템 검색(PostGIS DB 연산)
- **사용처**: 프로젝트 전체 (`src/lib/supabase.ts`)
- **상태**: 연동 및 스키마 세팅 완료

---

### 📷 [NEW] 4. HTML5 WebRTC & Device Orientation API (라이트 AR 뷰어용) - [연동 예정 ⏳]

- **용도**: Unity ARCore를 대체하여, 별도 앱 설치 없이 스마트폰 웹 브라우저(`app.gumi-ramen.com`)에서 후면 카메라를 켜고, 폰의 기울기(자이로스코프 센서)에 맞춰 허공에 라면 재료(이미지)를 둥둥 띄우는 **웹 AR 효과** 구현.
- **특징**: 구글 등 외부 개발사 API 키 발급이 전혀 필요 없는 **무료 HTML5 순수 웹 표준 기술**입니다.
- **사용처**: 앞으로 만들 B2C 모바일 화면 (`/app/camera` 라우터 등)
- **진행 내역**: 이제 이 API를 활용하여 모바일 클라이언트(유저 화면) 코딩을 시작합니다.
